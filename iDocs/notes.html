Documentation For Sudoku-Crypto

About

Sudoku-Crypto is a small project that was designed by the Attachees from the University of Nairobi at GearBox. The project had 2 sides. The server side and the user side. In the server side, a Node MCU was used to generate the sudoku puzzles and send notifications when a person solved the sudoku. The server side was programmed using Arduino language which has syntax very similar to the C syntax. On the front end an Android application was used to display the sudoku puzzle to be solved and also facilitate interaction with the Node MCU. Flutter was used due to its versitility. For the communication between the android phones and the server UDP was used. The team got to learn how to work with these technologies as for most of us this was the first time using these technologies. The code for the front end can be viewed here. My role in the project was writing the code for communication from the mobile app and also part of the sudoku class. I shall speak on these here.

UDP Communication In Dart

UDP communication in dart can be achieved using the dart library 'dart:io''. It contains access to classes required for accessing sockets that are used during internet connection. Below is a code snippet from the project. It is the class that was used to handle the communication from the android devices. Many of the functions are asynchronous thus Futures were required. For this reason, the dart library 'dart:async' was also required. 


            class packetHandler {
              int portNumber = 50000;
              String? destIpString;
              late RawDatagramSocket
                  socketConnection; //the socket connection that will be carrying out the udp message sending
            
              packetHandler();
            
              Future < void > initializeIp(int number) async {
                this.socketConnection =
                    await RawDatagramSocket.bind(InternetAddress.anyIPv4, number);
                this.portNumber = number;
                this.destIpString = "192.168.43.235"; //hard coded value contains ip of server
              }
            
              void sendData(String stringToSend) {
                List < int > buffer = ascii.encode(stringToSend);
                InternetAddress destAddress = InternetAddress(this.destIpString!);
                socketConnection.send(buffer, destAddress, this.portNumber);
              }
            }

UDP communication in dart is achieved using datagram sockets. In particular the class RawDatagramSocket is used to achieve this. the RawDatagramSocket is bound to an IP address and a port number using the code RawDatagramSocket.bind(Ip,port). As this requires both an IP address and a port number, it was seen not to require it to be done during instantiation. In order to send a UDP packet, the RawDatagramSocket requires a destination IP address, destination port number and Ascii encoded string.
